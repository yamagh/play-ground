# Play Framework & Svelte チュートリアル (新版)

静的なページの作成から始め、データベースと連携した動的な一覧表示（Read）、登録・更新（Create/Update）、削除（Delete）といった、Webアプリケーションに不可欠なCRUD機能一式を段階的に実装していきます。
